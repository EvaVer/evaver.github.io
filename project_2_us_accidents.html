Car Accidents in the US

A dashboard that provides insights about locations and weather types associated with car accidents

This project is still under development! I have started this, but i haven't finished putting the project on this web page.

Description

This project was executed with a car insurance company in mind. This dashboard provides the insurance company with insights regarding the location, time, and weather conditions that are associated with the number and severity of car accidents.

  • !

    A dashboard is an interactive overview of important metrics. A user can simply use the overview of the dashboard to gather basic information, but they can also choose to select variables and gather deeper insights based on their needs. Ultimately, dashboards can support data-driven decision-making.

Data Preparation

Before creating the dashboard, the data needs to be cleaned and wrangled into a usable format. Different datasets require different processing steps, but important steps include removing duplicate data, handling missing data, and checking values and data types.

Results

This dashboard gives its end-users — employees of a car insurance company — more insights into variables that can influence the risk of accidents and the severity of accidents. This can help them estimate the costs of accidents and supports decision-making towards new premiums.
